# 🔮 MirrorFish 舆情推演模拟器

> AI驱动的游戏舆情多Agent推演系统

## 功能特性

- 🧠 **和平精英游戏知识库**：传世武器、皮肤体系、地铁逃生等核心游戏机制
- 👥 **68类Agent画像**：724个虚拟玩家，覆盖竞技/商业化/社交/内容创作等维度
- 🌐 **多平台分区仿真**：微博/贴吧/小红书/抖音/B站，每个平台有不同的发言风格
- ⏰ **时间线仿真**：自动/实时爆发/慢燃型/预告型 4种模式
- 📊 **可视化分析**：实时危机指数、情绪分布、情绪走势图、阵营分布、热词TOP5
- 📝 **自动生成报告**：总体评估、核心风险点、防御建议、48小时走势预判
- 🔥 **8个预设场景**：传世武器返场、SS39赛季更新、反外挂大升级等

## 技术架构

- 纯前端运行，无需后端服务
- 规则引擎 + 预置语料库 + 情绪传播模型
- KOL影响力模型 + 沉默大多数跟风机制
- 轮次递进：初始→发酵→高潮→扩散→固化
